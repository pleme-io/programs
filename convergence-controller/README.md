# convergence-controller

Reconciles `Process` CRs through the 8-phase Unix-process lifecycle,
emits FluxCD `Kustomization`/`HelmRelease` resources for downstream
controllers, computes three-pillar BLAKE3 attestation chains.

**Replaces** [`tatara/tatara-reconciler`](https://github.com/pleme-io/tatara/tree/main/tatara-reconciler)
once Phase β–δ from
[`theory/WASM-RUNTIME-COMPLETE.md` §III.2](https://github.com/pleme-io/theory/blob/main/WASM-RUNTIME-COMPLETE.md)
complete. **Closes the recursive loop** — the platform's reconciler
is itself a tatara-lisp program running on the platform.

## Authoring shape

V.3 from
[`LISP-YAML-CONTROLLERS.md` §V](https://github.com/pleme-io/theory/blob/main/LISP-YAML-CONTROLLERS.md):
50/50 — typed CRD spec for operator-facing config, Lisp body for the
state-machine logic. Composes two macros from the controllers
library:

- `defstate-machine` — declares the 8-phase transition table.
- `defrule-driven-controller` — emits the CRD, informers, dispatch
  loop, status propagation.

## Migration phasing

- **Phase α (now)**: this `.tlisp` is reviewed but not deployed.
- **Phase β**: Rust `tatara-reconciler` adds `spec.runtime: rust|lisp`.
  Operators opt in per-Process CR.
- **Phase γ**: ~30 days of parallel-run parity → default to `lisp`.
- **Phase δ**: deprecate the Rust controller. The
  [`lareira-tatara-reconciler` chart](https://github.com/pleme-io/helmworks)
  drops the Rust image; `tatara-system` runs *only* this `.tlisp`.

## Why bootstrap-recursive

```
1. substrate/tool-image-flake builds the wasm-operator image.
2. wasm-operator pulls itself + the wasm-engine image from ghcr.io.
3. wasm-operator boots; reads ComputeUnit CRs.
4. ONE of those CRs is convergence-controller.tlisp.
5. The wasm-operator instantiates a wasm-engine running this Lisp.
6. The Lisp watches Process CRs.
7. ONE of those Process CRs declares: "deploy lareira-wasm-platform
   per the manifest at github:pleme-io/k8s/clusters/<name>/infrastructure/wasm-stack/"
8. The Lisp emits a Kustomization CR; FluxCD applies it.
9. The applied Kustomization redeploys the wasm-operator from the new
   tagged image.

The platform reconciles itself. Lisp closes the loop on Rust.
```
